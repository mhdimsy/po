# CODEX START HERE

تو قرار است یک **Full Digital Twin Prototype** برای دمو و شبیه‌سازی تولید کارخانه بسازی.

## قبل از هر کاری

اول این فایل‌ها را بخوان:

```text
docs/CODEX_OPERATING_RULES.md
docs/DOCUMENTATION_POLICY.md
docs/reference/CONTEXT.md
docs/reference/DECISIONS_SUMMARY.md
docs/reference/ARCHITECTURE.md
docs/reference/DOMAIN_MODEL.md
tasks/TASKS_INDEX.md
```

## قانون شروع

فعلاً کدنویسی نکن.
بعد از خواندن فایل‌ها، فقط این خروجی را بده:

1. خلاصه فهمت از پروژه در ۱۰ bullet
2. تکنولوژی‌های قطعی
3. چیزهایی که در V1 نباید پیاده شوند
4. ریسک‌های اصلی
5. پیشنهادت برای اجرای `TASK-001`
6. فایل‌ها و پوشه‌هایی که در `TASK-001` می‌سازی

تا وقتی کاربر تأیید نکرده، وارد کدنویسی نشو.

## قانون اجرای تسک‌ها

هر بار فقط یک فایل تسک از پوشه `tasks/` اجرا کن.
اگر کاربر گفت «تسک بعدی»، فایل بعدی را بخوان و فقط همان را انجام بده.

# Codex After Each Task Checklist

بعد از اتمام هر تسک، این چک‌لیست را اجرا کن:

## 1. Summary

- چه چیزی ساخته شد؟
- چه چیزی عمداً ساخته نشد؟

## 2. Files changed

لیست فایل‌ها و نقش هرکدام.

## 3. Run instructions

دستور اجرای backend/frontend/migration/test.

## 4. Tests

چه تستی انجام شد و نتیجه چه بود؟

## 5. Documentation updates

حداقل این‌ها را آپدیت کن:

```text
logs/IMPLEMENTATION_LOG.md
logs/CHANGELOG.md
logs/TESTING_LOG.md
docs/PROJECT_STATUS.md
```

در صورت نیاز:

```text
logs/DB_SCHEMA_CHANGE_LOG.md
logs/KNOWN_ISSUES.md
logs/OPEN_QUESTIONS.md
docs/modules/<module>.md
adrs/ADR-XXX.md
```

## 6. Next task

پیشنهاد بده تسک بعدی چیست، اما آن را شروع نکن.
